- 간단한 사진 보내기 버튼 추가
- 시연

  - 이미지 클릭하여 전체 이미지 보기

# 추가 기능 - 채팅방에서 비디오 보내기 및 보기

- 비디오 메시지 한 개 보내기
- 비디오 메시지는 비디오의 썸네일로 렌더링됨
- 비디오를 클릭하면 비디오를 재생할 수 있음
- 비디오 메시지는 DB에 저장되어야 함
- 간단한 비디오 보내기 버튼 추가

- 시연
  - 아무 사용자에게 비디오 1개 보내기
  - 비디오 클릭하여 비디오 재생하기

# 추가 기능 - 새 메시지가 올 때 푸시 알림

- 시연
  - 다른 측에서 새 메시지 보내고 푸시 알림 보여주기
